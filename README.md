# VoMo USSD Simulator

A Python simulation of a mobile money USSD menu system.

## How to run
1. Make sure Python is installed
2. Run VOMO_USSD_SIMULATOR.py
3. Dial *188# or type p to start

## Features
- Interactive USSD menu
- Transfer money
- Momo Pay & Pay Bill
- Airtime & Bundles
- Allow Cash Out
- Financial services
- Error handling with try/except
- Realistic time delays
